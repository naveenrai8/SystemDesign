# Url Shortener Service

Url Shortener service is being used to generate the Short URL/Alias of a given long url. This short URL can be used in following scenarios:
1. Easy to share it with Clients as it saves space.
1. Hide the long URL (in case of Affiliated URLs)

# Version 1

Design a service which does the following:
1. Generate Short Url from Long Url.
1. Retrieve Long Url from Short Url.

Non Functional Requirement:
1. It provides minimal latency possible.
1. High Availability of the system

Out of the Scope:
1. Deploy to multi-region.
1. Matching the latency between intra-region vs inter-region.
1. Enable REST API
